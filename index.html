وبسایت نمایشگاه ماشین

پروژه طراحی وبسایت نمایشگاه ماشین که شامل بخش‌های متنوعی از جمله نمایش موجودی خودروهای آمریکایی و آلمانی، گالری تصاویر، نظرات مشتریان، فرم تماس و بخش ویژه خودروهای خاص است. این وبسایت با طراحی مدرن و واکنش‌گرا، قابلیت فیلترینگ خودروها بر اساس نوع و نمایش جزئیات کامل هر خودرو را ارائه می‌دهد.

پنل مدیریت

ین پروژه یک پنل مدیریتی وب‌محور است که برای مدیریت کاربران، محتوا و تنظیمات سیستم طراحی شده. با استفاده از فناوری‌های مدرن مانند Bootstrap و FontAwesome، رابط کاربری کاربرپسندی ارائه می‌دهد. پشتیبانی از زبان فارسی و جهت راست‌به‌چپ (RTL) دارد.

وب‌سایت خبری ارزهای دیجیتال

این پروژه یک وب‌سایت ایستا و جذاب برای پوشش اخبار و به‌روزرسانی‌های حوزه ارزهای دیجیتال است. شامل بخش‌هایی مانند اخبار اخیر، اخبار استخراج کریپتو، پست‌های محبوب، و اطلاعات بازار بیت‌کوین می‌شود. با طراحی مدرن و تصاویر مرتبط، کاربران را با آخرین تحولات بازار، تحلیل‌ها، و رویدادهای مرتبط با بلاکچین و کریپتوکارنسی آگاه می‌کند.

فرم ورود شیشه‌ای

این پروژه یک فرم ورود زیبا و مدرن با طراحی شیشه‌ای (Glassmorphism) است که از کتابخانه Bootstrap برای ریسپانسیو بودن استفاده می‌کند. فرم شامل فیلدهای ایمیل و رمز عبور، گزینه یادآوری رمز عبور، و لینک‌های فراموشی رمز عبور و ثبت نام است. پس‌زمینه تصویر دارد و از آیکون‌های Font Awesome استفاده می‌کند. مناسب برای وب‌سایت‌های فارسی‌زبان است.

داشبورد تحلیل عملکرد سایت‌ها

این پروژه یک صفحه وب ساده است که با استفاده از HTML و CSS ساخته شده. جدول اصلی آمار عملکرد چندین سایت محبوب مانند گوگل، توییتر، آمازون و غیره را نمایش می‌دهد، شامل تعداد بازدیدها (views)، کلیک‌ها (clicks) و زمان متوسط حضور کاربر (average time). طراحی با تم تاریک انجام شده و شامل افکت‌های تعاملی مانند تغییر رنگ هنگام هاور روی سلول‌ها است. این پروژه برای نمایش داده‌های تحلیل سایت‌ها در یک فرمت جدولی و جذاب طراحی شده است.
